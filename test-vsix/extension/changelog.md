# Change Log

All notable changes to the "Neon Storm" theme will be documented in this file.

## [0.0.1] - 2026-03-29

### 🎉 Initial Release

- **Ultra-vivid color scheme** with maximum saturation for high-impact syntax highlighting
- **Full Angular support**: HTML templates, TypeScript, SCSS/CSS with optimized highlighting
- **Full .NET support**: C# with perfect attribute, type, and method highlighting
- **Neon UI accents**: Electric cyan (`#00DFFF`) active tab borders and cursor
- **Deep dark backgrounds**: Editor (`#16161E`), Sidebar (`#1A1B26`), Terminal (`#121217`)
- **Real screenshots included**: TypeScript, C#, HTML, and SCSS examples showcasing the theme

#### Syntax Colors
- Keywords: `#F7768E` (Magenta/Pink NEON)
- Functions/Methods: `#E0AF68` (Gold BRIGHT)
- Types/Classes: `#73DACA` (Turquoise ELECTRIC)
- Variables: `#7AA2F7` (Blue VIVID)
- Strings: `#9ECE6A` (Green LIME)
- Decorators/Attributes: `#BB9AF7` (Purple ELECTRIC)
- Numbers: `#FF9E64` (Orange VIBRANT)
- Comments: `#4A5F4A` (Green dark)

#### Language-Specific Features
- **HTML**: Tags in pink, attributes in purple, Angular directives highlighted
- **TypeScript**: Decorators (`@Injectable`, `@Component`) in purple, types in turquoise
- **C#**: Attributes (`[Route]`, `[HttpGet]`) in purple, types in turquoise
- **CSS/SCSS**: Selectors in turquoise, properties in blue, values in gold
- **JSON**: Keys in turquoise, strings in green, numbers in orange
- **Markdown**: Headings in pink, links in turquoise, code in blue

#### UI Features
- High-contrast selection highlighting
- Terminal colors matching syntax palette
- Vibrant error/warning/info colors
- Clear git decoration colors
- Semantic highlighting enabled for TypeScript and C#
